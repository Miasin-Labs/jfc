# Google AI Studio - Complete REST API Endpoint List
# Source: JS deobfuscation + Burp capture + live testing
# Date: 2026-05-21

## Public REST API (generativelanguage.googleapis.com)
## Auth: ?key=<GEMINI_API_KEY> or Authorization: Bearer <key>

### Model Management
GET    /v1beta/models                              # List all models
GET    /v1beta/models/{model}                      # Get model metadata
GET    /v1/models                                  # Stable (fewer models)
GET    /v1alpha/models                             # Alpha (same as v1beta)

### Content Generation
POST   /v1beta/models/{model}:generateContent      # Non-streaming generation
POST   /v1beta/models/{model}:streamGenerateContent?alt=sse  # SSE streaming
POST   /v1beta/models/{model}:batchGenerateContent # Batch (multiple requests)
POST   /v1beta/models/{model}:countTokens          # Token counting
POST   /v1beta/models/{model}:generateContentPerUserQuota  # Quota-aware

### OpenAI-Compatible (Authorization: Bearer <key>)
GET    /v1beta/openai/models                       # List models (OpenAI format)
POST   /v1beta/openai/chat/completions             # Chat completions
POST   /v1beta/openai/embeddings                   # Embeddings
POST   /v1beta/openai/images/generations           # Image generation
POST   /v1beta/models/{model}:chatCompletions      # Per-model chat

### Embeddings
POST   /v1beta/models/{model}:batchEmbedContents   # Batch embeddings
POST   /v1beta/models/{model}:embedContent         # Single embedding

### Caching
GET    /v1beta/cachedContents                      # List cached contents
POST   /v1beta/cachedContents                      # Create cached content
GET    /v1beta/cachedContents/{id}                 # Get cached content
PATCH  /v1beta/cachedContents/{id}                 # Update
DELETE /v1beta/cachedContents/{id}                 # Delete

### File Management
GET    /v1beta/files                               # List uploaded files
POST   /v1beta/files                               # Upload file
GET    /v1beta/files/{id}                          # Get file metadata
DELETE /v1beta/files/{id}                          # Delete file

### Tuned Models
GET    /v1beta/tunedModels                         # List tuned models (501 currently)
GET    /v1beta/tunedModels/{id}                    # Get tuned model
POST   /v1beta/tunedModels                         # Create tuned model

### Corpora (v1alpha - RAG)
GET    /v1alpha/corpora                            # List corpora
POST   /v1alpha/corpora                            # Create corpus
GET    /v1alpha/corpora/{id}                       # Get corpus
DELETE /v1alpha/corpora/{id}                       # Delete corpus
POST   /v1alpha/corpora/{id}/documents             # Add document
POST   /v1alpha/corpora/{id}:query                 # Query corpus (RAG)

### Operations
GET    /v1beta/operations                          # List async operations
GET    /v1beta/operations/{id}                     # Get operation status

### Vertex-style (limited availability)
POST   /v1beta/models/{model}:predict             # Vertex predict format
POST   /v1beta/models/{model}:predictLongRunning  # Long-running predict
